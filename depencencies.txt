This program depends on some external Packages. All of them are free software under a copyleft licence.

1: FFmpeg
    LICENCE: LGPL 2.1+
    Available in all GNU/Linux package managers I know of under the name 'ffmpeg'. Examples:
    apt-get install ffmpeg
    pacman -S ffmpeg

2: unoconv
    LICENCE: GPLv2
    Available in all GNU/Linux package managers I know of under the name 'unoconv'. Examples:
    apt-get install unoconv
    pacman -S unoconv

    2.1: LibreOffice
        LICENCE: MPL 2.0 (Mozilla Public Licence)
        Unoconv depends on LibreOffice/OpenOffice (LibreOffice is recommended)
        Already preinstalled in most distros, including Ubuntu and Linux Mint
        Available in all GNU/Linux package managers I know of under the name 'libreoffice'. Examples:
        apt-get install libreoffice
        pacman -S libreoffice

3: Handbrake
    LICENCE: GPLv2
    Available on github.com/HandBrake/HandBrake and on handbrake.fr
    The Converter comes with the Windows CLI version because I *really* dont feel like using Flatpaks here.
    If you get it working reliably with the Flatpak, please open a pull request.

    3.1: Wine
        LICENCE: LGPL 2.1+
        Available in all GNU/Linux package managers I know, but under varying names.
